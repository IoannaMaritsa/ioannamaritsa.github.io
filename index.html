Project completed withing the framework of the "Human-Computer Interaction" Course

Project completed within the framework of the "Internet Application Technologies" Course

The objective was to develop a website offering functionality similar to that of LinkedIn, a professional networking platform. The site should facilitate career-oriented services such as building professional profiles, connecting with other professionals, sharing job opportunities, showcasing skills and achievements etc.

The UI was implemented using the React framework and our database was created using Supabase.
